# Mod Commands

This chat commands gives you the option to enable some simple commands to help you manage the stream from Twitch chat, here is how to use them:

---

| **Command**     | **Description**                             | **Example**   |
|-----------------|:--------------------------------------------|:--------------|
| `sourceinfo`    | gives you details about the SOURCE in chat. | `!sourceinfo` |
| `serverinfo`    | gives you details about the SERVER in chat. | `!serverinfo` |
| `fix` / `f`     | tries to fix the stream.                    | `!fix`        |
| `refresh` / `r` | tries to refresh the stream.                | `!refresh`    |
| `bitrate` / `b` | returns the current bitrate.                | `!bitrate`    |
